Quiz Cheirosa

Projeto desenvolvido com fins de aprendizado de tecnologias web

O Quiz Cheirosa é um jogo de perguntas e respostas baseado em letras de músicas dedicado à minha namorada. São 12 perguntas ao total e cada pergunta possui até 3 tentativas.


## Tecnologias utilizadas

- React (Vite)
- Node.js
- Express
- ESLint


## Como rodar o projeto localmente

Pré-requisitos
- Node.js (versão 18 ou superior)
- npm

Passo a passo
* instalar dependências
npm install

* build do frontend
npm run build

* iniciar servidor
node server.js

Acesse http://localhost:3000
